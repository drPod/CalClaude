App Icon — Blended Concepts

One unified element instead of two competing ones

Blend 1: Terminal Running `cal`

A terminal window showing the output of the Unix `cal` command. The calendar IS the terminal output. Today highlighted. One unified metaphor — it's both a terminal and a calendar.

Blend 2: Calendar with Cursor

A calendar icon where today's cell has a blinking text cursor inside it — as if you're typing directly into the calendar. The CLI is implied by the cursor, not a separate element.

Blend 3: Calendar Page as Prompt

A calendar page where the body IS a command prompt — the >_ sits where the date number normally would. Purple gradient, clean. The calendar IS the interface.
